# @vuepress/plugin-google-analytics (opens new window)

Google analytics plugin

# Install

yarn add -D @vuepress/plugin-google-analytics
# OR npm install -D @vuepress/plugin-google-analytics

WARNING

We currently recommend using Yarn (opens new window) instead of npm to install all dependencies if you are using Google Analytics Plugin, because npm fails to generate the correct dependency tree in this case.

# Usage

module.exports = {
  plugins: [
    [
      '@vuepress/google-analytics',
      {
        'ga': '' // UA-00000000-0
      }
    ]
  ]
}

TIP

Please be aware of GDPR (2018 reform of EU data protection rules) (opens new window) as IPs are anonymized automatically.

# Options

# ga

  • Type: string
  • Default: undefined

Provide the Google Analytics ID to enable integration.
